# Project-Bank-Account-Simulator

This project demonstrates the evolution of a Java application from a basic Maven setup to a Hibernate-based implementation, and finally to an optimized version.
Each stage is maintained in a separate Git branch to showcase the improvements in structure, performance, and database handling.


## Project Versions

### 1. Basic Maven Project
- Branch: `maven-version`
- Description: Initial setup using Maven with basic project structure.
- Features:
  - Dependency management with Maven
  - Simple Java application

### 2. Hibernate Version
- Branch: `hibernate-version`
- Description: Integrated Hibernate ORM for database operations.
- Features:
  - Entity mapping
  - Hibernate configuration
  - CRUD operations

### 3. Optimized Version
- Branch: `optimized-version`
- Description: Improved performance and code structure.
- Enhancements:
  - Better session management
  - Reduced redundancy
  - Improved query handling
 
## Technologies Used
- Java
- Maven
- Hibernate ORM
- PgSQL
- Git

## Key Learnings
- Understanding Maven project structure
- Implementing ORM with Hibernate
- Performance optimization techniques
- Managing multiple versions using Git branches
